SLAC @ Olin College

SLAC is a group to help Olin students gain skills to work on awesome projects. We teach introductory programming, provide resources and tutorials, and help promote Oliner's personal and collaborative projects to continue our school's amazing culture of creation.

Olin Resources

Hosting

We provide or recommend a few resources to students for hosting sites online. If you've never hosted a web page before, we can get you started.

  • Free hosting with FTP/shell access. Provided by Dreamhost. These servers support running PHP scripts and MySQL databases. Email Tim Ryan for more information.
  • Github static hosting. If you have a Github account, look into Github Pages. You can host a site on Github that points to any of the Olin domains below.

  • Heroku/cloud services. Plenty of cloud services have free tiers you can use as a student to get started, and run platforms like Python, Ruby, or Node.js
